# Spot-USD-landing-page

## Contains
- React
- Redux
- webpack
- Babel
- PostCSS
- React Intl

## Step
```
$ yarn install
```

## Running
```
$ yarn run serve
```

## Testing
```
$ yarn test
```

## Build

### development
```
$ yarn dev-build
```

### staging
```
$ yarn run staging-build
```

### production
```
$ yarn run prod-build
```
